whereCanIJump

  • whereCanIJump(board: I.IBoard, from: IPosition, positions?: IPosition[]): IPosition[]
    • Defined in Board.ts:220

    Gets all positions where can I jump recursively.

    1. Get not empty near positions from board.
    2. Foreach not empty near position:
      • Get jump position.
      • If jump position do NOT exists or accumulated positions contains jump position then return accumulated positions.
      • Set Jumping black piece to true if is black piece.
      • Set Jumps to from + from.jumps.
      • Call and return this method again recursively to get next jump positions.

    Parameters

    • board: I.IBoard
    • from: IPosition
    • Optional positions: IPosition[]

    Returns IPosition[]
